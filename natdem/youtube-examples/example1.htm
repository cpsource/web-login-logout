Resizable Box Example

This example demonstrates a resizable container. You can drag the bottom-right corner of the box to resize it.

This is some text inside a resizable container. You can resize this box by dragging the handle at the bottom-right corner. As you resize the box, the content inside will adjust accordingly, and scrollbars will appear if the content exceeds the new dimensions.
